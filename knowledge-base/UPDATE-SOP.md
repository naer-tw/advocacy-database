# 知識庫更新作業流程（SOP）

> 確保知識庫與最新發稿同步

## 📅 更新頻率

| 類型 | 頻率 | 負責 |
|------|------|------|
| 新聞稿 | **每次發稿後** | 發稿者 / AI 助手 |
| 政策立場 | 立場改變時 | 理事長確認 |
| FAQ | 每季檢視 | AI 助手 |
| 年度報告 | 每年底 | 秘書處 |

---

## 🆕 新增新聞稿流程

### 1. 確認定稿版本
- 確保是**最終定稿**（非草稿）
- 檔名格式：`YYYYMMDD-標題.md` 或 `YYYYMMDD國教行動聯盟新聞稿.md`

### 2. 存放位置
```
press-complete/
└── 2026/
    └── 03月/
        └── 20260324-新聞稿標題.md
```

### 3. 更新 Google Sheets
- 開啟 [新聞稿清單](https://docs.google.com/spreadsheets/d/1E_XELmjuPZ5NEQCwOfMZOD9sRqbsXTwYB98tUJlq2_8)
- 新增一行，填寫：日期、標題、連結、新聞露出、議題分類、議題標籤

### 4. 同步到 GitHub（如有權限）
```bash
cd ~/clawd/projects/npo-consulting/knowledge-base/naer
git add .
git commit -m "新增 YYYY-MM-DD 新聞稿：標題"
git push
```

### 5. 通知 AI 助手（可選）
告訴妃寶：「知識庫新增了一篇新聞稿：[標題]」

## 🏷️ 議題標籤規範

### 六大分類（必選一個）
1. 校園安全與管理
2. 菸害與藥物防制
3. 教育政策與改革
4. 性健康與性教育
5. 兒童保護與權益
6. 心理健康與神經多樣性

### 常用標籤（可複選）
| 分類 | 標籤 |
|------|------|
| 校園安全 | 司法少年、校園霸凌、校園安全、3C成癮、手機管理 |
| 菸害 | 加熱菸、電子煙、喪屍煙彈、加味菸、藥物濫用 |
| 教育政策 | 升學制度、教師權益、營養午餐、食品安全 |
| 性健康 | 性教育改革、青少年性病、性影像防護 |
| 兒童保護 | 兒少權法、兒虐防治、兒童權利公約、兒家專責單位 |
| 心理健康 | 神經多樣性、情緒障礙、早期療育、兒少輕生 |

## ⚠️ 注意事項

### 不要放入知識庫的內容
- ❌ 草稿版本
- ❌ 內部討論文件
- ❌ 未公開的策略文件
- ❌ 其他組織的新聞稿（除非聯名）

### 檔案命名規則
- ✅ `20260324-國教盟呼籲修法.md`
- ✅ `20260324國教行動聯盟新聞稿.md`
- ❌ `新聞稿草稿v3.md`
- ❌ `給瀚瀚看的版本.md`
