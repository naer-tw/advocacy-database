# 國教盟 2025 年度成果

## 摘要

2025 年，國教行動聯盟累計獲得 **2,040 則新聞報導**，成功推動多項政策改善。以下是年度 20 大關注議題及成果。

---

## 年度統計

| 項目 | 數據 |
|------|------|
| 新聞報導數 | 2,040 則 |
| 時間範圍 | 2024/12/30 - 2025/11/02 |
| 舉辦記者會 | 多場 |
| 政策採納 | 多項 |

---

## 年度 20 大議題成果

### ✅ 成功推動的政策

| 議題 | 成果 |
|------|------|
| 🚬 加熱菸審查管制 | 衛福部按國教盟建議進行管制 |
| 🩺 青少年性病防治 | 衛福部與教育部均有新防治計畫 |
| 💨 喪屍煙彈管制 | 因國教盟努力，升級至二級毒品 |
| 🧠 神經多樣性正名 | 心健司、教育部承諾推動去污名化 |
| 🍼 班班喝鮮乳 | 各縣市響應，營養午餐免費政策推出 |
| 📚 性教育改革 | 教育部已有新計畫 |
| 👨🏫 教師離線權 | 陸續有縣市開始倡議 |
| 📖 私校小聯考 | 教育部已有回應 |
| 😢 兒少輕生防治 | 與兒童健康聯盟合辦論壇 |
| 🍽️ 食安把關 | 與食力雜誌合辦論壇並發佈調查 |
| 📊 青少年十大議題 | 召開三場記者會，教育部有回應 |

### 🔄 持續推動中

| 議題 | 進度 |
|------|------|
| 📱 校園手機集中管理 | 已向教育部提出建議，持續推動 |
| 🚫 禁止加味菸 | 衛福部長承諾改善 |
| 👶 兒虐致死修法 | 立委已在推動修法 |
| 🏥 早期療育量能 | 衛福部承諾改善 |
| 🏫 校園安全專責單位 | 教育部設立副組長，持續倡議設立專責單位 |
